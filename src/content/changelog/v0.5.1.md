## New Features

### Numerical Calculus Module

- Differentiation (numerical derivatives)
- Integration (trapezoid, Simpson, Romberg)
- Root finding (bisection, Newton-Raphson, secant)
- High-precision numerical methods

### Dependency Injection

- `LambdaEvaluator` trait for clean architecture
- Decoupled evaluation logic
- Testable components
- Extensible design

### WASM Bindings

- 10 new numerical function exports
- WebAssembly integration
- Browser-compatible API
- Performance optimizations

### TypeScript SDK

- `NumericalOps` module
- Type-safe interfaces
- Async/await support
- Error handling
